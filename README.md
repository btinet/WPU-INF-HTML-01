### HTML Grundlagen
Beispiel einer HTML-Datei:
````html
<!-- index.html /-->
<html>
    <head>
        <meta charset="UTF-8">
        <title>Dokumententitel</title>
    </head>
    <body>
        <h1>Tabellenüberschrift</h1>
        <p>Absatz</p>
    </body>
</html>
````
